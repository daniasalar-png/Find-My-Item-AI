# Find-My-Item-AI
AI-powered Lost &amp; Found Management System built using Java Swing and MySQL.
## Features

- User Registration & Login
- Report Lost Items
- Report Found Items
- Search Items
- Manage Items
- AI-based Item Matching
- Edit/Delete/Mark Returned
- Modern Swing GUI

## Technologies Used

- Java
- Java Swing
- MySQL
- JDBC

## Database

MySQL
